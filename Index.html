Finding a Channel ID

YouTube channel IDs start with UC and are 24 characters long. Here's how to find one:

  1. Go to the YouTube channel you want to add.
  2. Look at the URL bar. If it shows /channel/UCxxxxxx, that's the ID — copy everything after /channel/.
  3. If the URL shows a custom handle like /@ChannelName, click About on the channel page, then Share → Copy channel link. The link will contain the channel ID.
  4. Alternatively, right-click the channel page → View Page Source and search for "channelId".

The ID will look like: UCVHFbw7woebKtfvug_tMXOQ
